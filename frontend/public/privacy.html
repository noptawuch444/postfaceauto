นโยบายความเป็นส่วนตัว (Privacy Policy)

1. การเก็บรวบรวมข้อมูล

GoldSync AutoBot ("เรา") เข้าถึงข้อมูลพื้นฐานของท่านผ่าน Facebook API เมื่อท่านอนุญาตให้แอปพลิเคชันเข้าถึงเพจของท่าน ข้อมูลที่เราเข้าถึงได้แก่ ชื่อเพจ, รหัสเพจ (Page ID), รูปโปรไฟล์เพจ และ Page Access Token ทั่วคราว

2. การใช้งานข้อมูล

เราใช้ข้อมูลดังกล่าวเพื่อวัตถุประสงค์ในการให้บริการระบบตอบกลับอัตโนมัติ (Auto-Reply) และการจัดการโพสต์บนเพจเฟซบุ๊กตามที่ท่านตั้งค่าไว้เท่านั้น โดยระบบจะอ่านความคิดเห็นบนโพสต์และดำเนินการตอบกลับตามข้อความที่ท่านกำหนด

3. การคุ้มครองข้อมูล

เราเก็บรักษา Page Access Token ไว้อย่างปลอดภัยในฐานข้อมูลที่มีการเข้ารหัส และจะไม่นำข้อมูลของท่านไปเผยแพร่ ขาย หรือให้บุคคลภายนอกเข้าถึงโดยไม่ได้รับอนุญาต

4. การลบข้อมูล

ท่านสามารถยกเลิกการเชื่อมต่อเพจได้ทุกเมื่อผ่านหน้าจัดการเพจในระบบ หรือลบการอนุญาตผ่านการตั้งค่าแอปพลิเคชันใน Facebook เมื่อมีการลบข้อมูล ระบบของเราจะทำลาย Access Token ที่เกี่ยวข้องทันที

5. การติดต่อ

หากมีคำถามเกี่ยวกับนโยบายความเป็นส่วนตัวนี้ โปรดติดต่อผู้พัฒนาผ่านช่องทางที่ท่านได้รับลิงก์นี้
